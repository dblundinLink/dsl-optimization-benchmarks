# GA2Decay Scoring DSL Specification

## 1. OVERVIEW

The GA2Decay DSL defines scoring functions that compute per-neuron Help/Hurt scores from batches of (ordinal, signal) examples. Programs follow a constrained **weight→accumulate→combine→normalize** pipeline pattern, where newer examples receive higher weights via exponential decay. The DSL prevents arbitrary computations while enabling systematic exploration of decay rates, aggregation strategies, and combination modes to achieve ≥0.95 Pearson correlation with reference implementations.

## 2. OPERATORS

| Name | Args | Semantics | Example |
|------|------|-----------|---------|
| `DECAY` | `(type, rate)` | Returns weight function: `exp`→`e^(-rate*age)`, `linear`→`max(0,1-rate*age)`, `none`→`1.0` | `DECAY(exp, 0.03)` |
| `WEIGHT` | `(ordinal, decay_fn)` | Computes example weight from ordinal (age = max_ord - ordinal) | `WEIGHT(ord, decay)` |
| `HELP` | `(signal, weight)` | Returns `weight * max(0, signal)` per neuron | `HELP(sig, w)` |
| `HURT` | `(signal, weight)` | Returns `weight * max(0, -signal)` per neuron | `HURT(sig, w)` |
| `AGGREGATE` | `(values, op)` | Reduces across examples: `sum`, `mean`, `max` | `AGGREGATE(help, sum)` |
| `COMBINE` | `(help, hurt, mode)` | Merges scores: `diff`→`h-u`, `ratio`→`h/(h+u+ε)` | `COMBINE(h, u, diff)` |
| `NORMALIZE` | `(scores, method)` | Scales output: `none`, `zscore`, `minmax` | `NORMALIZE(s, none)` |

## 3. GRAMMAR

Programs follow this structure:
```
SCORE {
  decay  = DECAY(type, rate)
  FOR (ord, sig) IN batch:
    w = WEIGHT(ord, decay)
    help += HELP(sig, w)
    hurt += HURT(sig, w)
  h = AGGREGATE(help, op)
  u = AGGREGATE(hurt, op)
  result = NORMALIZE(COMBINE(h, u, mode), method)
}
```

**Parameter domains:**
- `type` ∈ {`exp`, `linear`, `none`}
- `rate` ∈ [0.0, 0.2] (float)
- `op` ∈ {`sum`, `mean`, `max`}
- `mode` ∈ {`diff`, `ratio`}
- `method` ∈ {`none`, `zscore`, `minmax`}

## 4. COMPILATION TARGET

The DSL compiles to a Python function with signature:

```python
def score(ordinals: np.ndarray, signals: np.ndarray) -> np.ndarray:
    """
    Args:
        ordinals: shape (batch_size,) - example indices (0 = oldest)
        signals: shape (batch_size, n_neurons) - per-neuron signals
    Returns:
        scores: shape (n_neurons,) - Help/Hurt score per neuron
    """
```

The function must be vectorized (NumPy) and handle batch_size=100. Age is computed as `max(ordinals) - ordinal` so newer examples (higher ordinal) have lower age and higher weight.
