# L-System DSL Specification

## 1. OVERVIEW

This DSL enables declarative specification of L-systems (Lindenmayer systems) for generating tree-like fractals via turtle graphics interpretation. It separates grammar definition (axiom, rewrite rules) from turtle semantics (drawing commands), supporting both deterministic and stochastic rule expansion. The DSL targets visual complexity optimization by exposing key parameters: branching angle, iteration depth, and length scaling. Programs compile to a `lsystem` Python function that returns a complete specification dictionary for rendering.

## 2. OPERATORS

| Name | Args | Semantics | Example |
|------|------|-----------|---------|
| `AXIOM` | `(str)` | Set initial string for rewriting | `AXIOM("X")` |
| `RULE` | `(sym, exp)` | Deterministic rewrite: sym → exp | `RULE("F", "FF")` |
| `STOCH` | `(sym, [(p,exp), ...])` | Probabilistic rule with weighted choices | `STOCH("X", [(0.6,"F[+X][-X]"), (0.4,"F[+X]")])` |
| `ANGLE` | `(deg)` | Set turtle turn angle in degrees | `ANGLE(25.7)` |
| `ITER` | `(n)` | Set iteration count (1-8) | `ITER(5)` |
| `SCALE` | `(factor)` | Length reduction per branch depth | `SCALE(0.85)` |
| `FORWARD` | `(sym)` | Register symbol as draw-forward | `FORWARD("F")` |
| `SKIP` | `(sym)` | Register symbol as no-draw (structural) | `SKIP("X")` |
| `MOVE` | `(sym)` | Register symbol as move without drawing | `MOVE("G")` |
| `TURN` | `(sym, dir)` | Map symbol to turn direction (L/R) | `TURN("+", R)` |
| `BRANCH` | `(push, pop)` | Define branch stack delimiters | `BRANCH("[", "]")` |
| `SEED` | `(n)` | RNG seed for stochastic reproducibility | `SEED(42)` |

## 3. GRAMMAR

Programs follow this structure: **header** → **rules** → **turtle mappings**

```
AXIOM("X")
ANGLE(25)
ITER(5)
SCALE(0.85)
SEED(123)

RULE("F", "FF")
STOCH("X", [(0.5, "F[+X][-X]FX"), (0.5, "F[-X]+X")])

FORWARD("F")
SKIP("X")
TURN("+", R)
TURN("-", L)
BRANCH("[", "]")
```

**Symbol alphabet**: Use uppercase letters for rewritable symbols. Reserved: `F`=forward, `G`=move, `+`=right turn, `-`=left turn, `[`=push state, `]`=pop state.

**Stochastic rules**: Probabilities should sum to 1.0. Multiple `STOCH` entries for same symbol are combined.

**Defaults**: If `FORWARD("F")` is omitted, `F` draws by default. If `BRANCH` is omitted, `[` and `]` are used.

## 4. COMPILATION TARGET

The DSL compiles to a Python function:

```python
def lsystem() -> dict:
    return {
        "axiom": str,           # Initial string
        "rules": dict,          # {sym: expansion} or {sym: [(prob, exp), ...]}
        "angle": float,         # Turn angle in degrees
        "iterations": int,      # Expansion count
        "scale": float,         # Length scale factor (default 1.0)
        "seed": int | None,     # RNG seed for stochastic rules
        "forward": set,         # Symbols that draw forward
        "skip": set,            # Symbols with no turtle action
        "move": set             # Symbols that move without drawing
    }
```

The returned dictionary fully specifies the L-system for a turtle graphics renderer.
